# TaskForge

TaskForge is a command-line task management application written in Python.

## Goals

- Practice object-oriented programming
- Build clean and modular Python applications
- Demonstrate software engineering principles
- Create a portfolio project

## Planned Features

- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Save/load tasks
- Search tasks
- Filter tasks
- Statistics

## Technologies

- Python
- JSON
- Git
